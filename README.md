## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/AbrarShafin/mi_card.git
    ```
2. Navigate to the project directory:
    ```bash
    cd mi_card
    ```
3. Install dependencies:
    ```bash
    flutter pub get
    ```
4. Run the app:
    ```bash
    flutter run
    ```
